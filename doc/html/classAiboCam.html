int AiboCam::updateMMap ( int  decompress = 1  ) 

Captures images from socket. Argument is used to toggle decompression of image. 1 = decompress 0 = no-decompress

If debug is set to true, the information about the image will be printed to stdout.

This method uses a slightly altered jpeg_decompress method with a larger buffer because we ran into buffer overflows with the stock version provided by Player.
